<p><i>TTS Util</i> is a utility Android app for synthesising text into audible speech.</p>
<p>Features:</p>
<ul>
<li>Read or create wave files from input text, clipboard text or text files using text-to-speech.</li>
<li>Configure app-specific text-to-speech settings: engine, voice, pitch and speech rate.</li>
<li>Custom silence for line endings, sentences, exclamations and questions.</li>
<li>Text filters for omitting certain text elements from the audio.</li>
<li>A playback-on-input mode for reading letters and words as they are typed.</li>
<li>Buttons for retrieving commonly read text.</li>
<li>Share menu entries and shortcuts for sharing input text to the app.</li>
</ul>
<p>Requirements:</p>
<ul>
<li>A device running Android version 4.0.3 or higher.</li>
<li>A separate Android text-to-speech engine app.</li>
</ul>
